### running with your images

<br>

* you can create your steganography images. first, install deps with:

```
pipenv install
````

* add `env` vars:

```
cp .env.example .env
vim .env
```

* add background and foreground images to their respective dirs (`background` and `foreground`). 

* finally, run:

```
./src/main.py
```
